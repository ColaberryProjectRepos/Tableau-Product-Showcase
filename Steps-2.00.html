Add a New User to the Site

Everyone who needs to access Tableau Online—whether to browse, publish, edit content or administer the site—must be added as a user.

Resource:  Add Users to a Site


1. Sign into Tableau Online: https://sso.online.tableau.com/public/idp/SSO  and select Users.


Username:   ali@colaberry.com 
Password:  Tableau2Colaberry

2. On the Users page, click Add Users, and then click Enter Email Addresses.


3. In the Enter email addresses box, enter the users’ email addresses. If you add more than one user, separate each address with     a     semicolon.
    
    For example, tdavis@example.com; jjohnson@example.com


            

4. Select the Explorer site role from the drop-down list, to assign that site role to all users you’re adding. 
5. For site role definitions, see  Set Users' Site Roles. 
6. Click Add Users.
